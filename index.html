Текст по центру

Текст по центру можно сделать так < center> < /center> [без пробелов], таким способом можно сделать по центру не только текст, но и картинки и многое другое!

Примеры:

1) < center> Текст < /center>

2) < center> < img src="Ссылка на картинку" < /center>

Ссылка под картинку/текст

Для этого используем данный код: < a href="">

Примеры:

1)< a href="Ссылка">Текст< /a>

2) < a href="ссылка">< img src="ссылка на картинку" < /a>
